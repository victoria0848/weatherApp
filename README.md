# weatherApp




Denne app viser vejret i Nordjylland med tre sider: Hjem (aktuelt vejr), Prognose (3 dages vejr) og Om (teknologier og links).  

Appen er lavet med **React**, **React Router**, **useEffect + fetch** og stylet med **Sass**. Layoutet er **mobile first** og semantisk korrekt.

For at køre appen:  
1. `npm install`  
2. `npm run dev`  
3. Åbn `http://localhost:5173` i browseren
